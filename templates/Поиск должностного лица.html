  • Поиск

    Поиск может осуществляться либо переходом в поиск организации либо в поиск должностного лица.


  • Поиск должностного лица:

    Поиск может быть осуществлен по следующим критериям, представленным в виде радио-кнопок:

    • ФИО должностного лица
    • Средства связи
    • Адрес
    • Комната
    • Должность
    • Дата рождения

    Выбрав критерий поиска, необходимо ввести текст в поисковую строку и нажать клавишу <Поиск>. На экране высвечиваются результаты поиска:

    • информация о количестве записей, отвечающих заданному условию поиска;
    • список должностных лиц, отвечающих критерию поиска

    По ссылкам можно выйти на:

    • Карточку должностного лица
    • Место его работы с указанием вышестоящей организации

    Если не нашлось записей, удовлетворяющих условиям поиска, выдается сообщение: "Ничего не найдено".

    Рекомендуем снова выбрать режим "Поиск" или по клавише "Изменить условия" сменить параметры поиска.

    Так как режим Поиска контекстный, то необходимо учесть ряд правил при заполнении поисковых показателей:

    • Если подразумевается "поиск по вхождению", то в конце или в начале поисковой строки необходимо указать "*" и в результат поиска попадут все записи, у которых есть такое вхождение знаков без учета регистра
    • Номера телефонов вводятся без использования дополнительных знаков
